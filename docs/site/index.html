Features
Everything a production gateway needs

Built phase by phase, each layer with a clear distributed systems reason for its position in the stack.

🔀

OpenAI-compatible routing

Drop-in replacement for the OpenAI API. Any SDK that speaks /v1/chat/completions works out of the box.

🧠

Semantic cache

Embeds prompts with nomic-embed-text and matches near-duplicates by cosine similarity (≥0.95). Cache hits return in 0.2ms.

⚖️

Load balancing

Round-robin across healthy backends using a lock-free atomic counter. Dead backends isolated within 5 seconds via health probes.

🔒

Rate limiting

Per-API-key token bucket implemented as a Redis Lua script — atomic across gateway replicas in a single round-trip.

⚡

Circuit breakers

Per-backend Closed→Open→Half-Open state machine. 3 failures trips the breaker, fast-rejecting for 30s before probing recovery.

🛡️

Backpressure

Bounded concurrency semaphore. When full, new requests get a 503 instantly — keeping p99 predictable under overload.

📊

Prometheus + Grafana

Latency histograms, cache hit rate, backend health gauges, circuit breaker state, and RPS — all scraped at /metrics.

☸️

Kubernetes ready

Multi-stage Docker build, K8s manifests, HPA (2→5 replicas), liveness and readiness probes wired to backend health.

🖥️

Live dashboard

Vanilla React dashboard polling /stats every 2s — backend health, cache hit rate, RPS sparkline, circuit breaker state.

Architecture
Request lifecycle

Every layer has a defensible reason for its position. Rate limit sits outside the queue — a rejected request never occupies a slot.

Quick start
Up in 3 commands

Requires Go 1.22+, Ollama, and Docker.

# 1. Pull models
ollama pull llama3.2:1b
ollama pull nomic-embed-text

# 2. Start Redis
docker run -d --name helmsman-redis -p 6379:6379 redis:7-alpine

# 3. Run the gateway
make run

# Optional: live dashboard
make dashboard   # → http://localhost:8081

# Optional: Kubernetes
make docker-build k8s-up k8s-status
